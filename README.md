# phamarcy-price-alamiese
this is useful for people trying to get used to prices set by their boss in a phamarcy store
Shelf Price — pharmacy price list & sales app

A small website so you (and anyone else at the counter) can search a medicine, tap the
right price (packet / box / card), ring up a sale, print a receipt, look back through
past sales, and route any price changes or new medicines through a suggestion box that
only you can approve or reject.
